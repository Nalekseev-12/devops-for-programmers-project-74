## Setup

### Requirements

* docker
* docker compose V2
* node >= 14
* make

**Run**

```bash
make build # build app
make ci # run tests
make start # run server
make push # push to dockerhub
```

**Dockerhub**

```bash
https://hub.docker.com/repository/docker/deathzzz/devops-for-programmers-project-74
```
